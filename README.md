<h2 id="objective">💡 | Objetivo</h2>

Esta página visa promover a visibilidade e credibilidade do **Mercadinho Aragão**, destacando pontos como:
 - Botão de contate-nos.
 - Avaliações reais tiradas do Google Maps.
 - Rota do Google Maps.
 - Fornecedores/Parceiros reais do comércio.
---
<br>
<h3> O intuito deste projeto é aplicar conhecimentos técnicos de UI/UX com o Figma, arquitetando a página e visando responsividade. </h3>

Seguintes competências aplicadas:

- Arquitetura organizada de arquivos internos.
- Tags semânticas visando SEO.
- Imagens responsivas.
- Media Queries.
